# 1. 이진 트리\(Binary Trees\)

## 재귀 함수\(Recursive functions\) 란?

* 하나의 함수에서 **자신을 다시 호출**하여 작업을 수행하는 것
* 왜 쓰이는가?
  * 생각보다 많은 종류의 문제가 재귀적으로\(recursively\) 해결 가능
