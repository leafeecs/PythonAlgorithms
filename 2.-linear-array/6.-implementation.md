## Insert element in a list

* L = \[20, 37, 58, 72, 91\] 이고 x = 65 인 경우,  올바른 리턴 값은 \[20, 37, 58, 65, 72, 91\] 

```python
def solution(L, x):
    less_than_x = []
    bigger_than_x = []
    for i in range(len(L)):
        if L[i] < x:
            less_than_x.append(L[i])
        else:
            bigger_than_x.append(L[i])
    answer = less_than_x + [x] + bigger_than_x
    return answer

L = [20, 37, 58, 72, 91]
x = 65
print(solution(L, x))
```

```python
def solution2(L, x):
    answer = []
    
    for i in range(len(L)):
        if L[i] < x:
            answer.append(L[i])
        else:
            continue

    answer.append(x)
    
    for i in range(len(L)):
        if L[i] > x:
            answer.append(L[i])
        else:
            continue

    return answer

L = [20, 37, 58, 72, 91]
x = 65
print(solution2(L, x))
```

* 같은 O\(n\) 이나, 리스트의 규모가 엄청 클 경우 solutin1 은 비효율적이다
